# 鞋子类目使用指南

## ✅ 概述

本爬虫支持 FARFETCH 的所有类目，包括女鞋类目。商品解析逻辑是通用的，可以正确提取任何类目的品牌、商品标题、价格、链接和图片。

---

## 📊 数据文件

### 女鞋类目 CSV

**文件位置：** `data/女鞋类目.csv`

**分类数量：**
- 一级分类：12 个
- 二级分类：26 个
- 总页数：约 700+ 页

**主要类目：**
1. Boots（靴子）- 机车靴、及膝靴、过膝靴、及踝靴、切尔西靴、工装靴、牛仔靴、雨靴和雪靴
2. 穆勒鞋 - 平底穆勒鞋、高跟穆勒鞋
3. 草编鞋 - 平底草编鞋、高跟草编鞋
4. 高跟鞋 - 平底鞋、高跟鞋
5. 凉鞋 - 平底凉鞋、高跟凉鞋
6. 运动鞋 - 低帮鞋、高帮鞋、套穿鞋
7. 德比鞋和牛津鞋
8. 人字拖和拖鞋
9. 乐福鞋
10. 拖鞋
11. 芭蕾平底鞋
12. 鞋履配饰

## 🚀 使用方式

### 方式一：快速测试女鞋类目

```bash
# 使用 Makefile
make quick-shoes

# 或使用命令行
python3 -m playwright_crawler.batch_crawler data/女鞋类目.csv
```

**说明：** 快速测试会爬取 3 个示例分类，大约需要 5-10 分钟。

---

### 方式二：完整爬取女鞋类目

```bash
# 使用 Makefile
make full-shoes

# 或使用命令行
python3 -m playwright_crawler.batch_crawler data/女鞋类目.csv
```

**说明：** 完整爬取会处理所有 26 个分类，预计需要 2-4 小时。

**注意：** 使用 `make full-shoes` 命令时，会要求确认以避免误操作。

---

### 方式三：自定义爬取范围

您可以编辑 CSV 文件，自定义爬取范围：

```csv
一级分类,二级分类,网页链接,从第几页开始,到第几页为止
Boots,机车靴,https://www.farfetch.com/cn/shopping/women/biker-boots-1/items.aspx,1,5
```

修改"从第几页开始"和"到第几页为止"列来控制爬取范围。

## 💡 技术说明

### 为什么可以直接使用女鞋类目？

1. **CSV 格式一致**
   - 女鞋类目 CSV 的格式与服装类目完全一致
   - 包含相同的列：一级分类、二级分类、网页链接、页码范围

2. **商品解析通用**
   - `parse_farfetch_product` 函数是通用的
   - 不依赖于特定的类目或商品类型
   - 可以正确解析 FARFETCH 任何页面的商品信息

3. **URL 格式一致**
   - 所有类目的 URL 格式相同
   - 爬虫逻辑适用于任何 FARFETCH 分类页面

## ⚠️ 注意事项

1. **网络稳定性**
   - 女鞋类目包含约 700+ 页数据
   - 建议确保网络稳定后再开始完整爬取
   - 可以分批次爬取，避免长时间运行

2. **时间估算**
   - 快速测试（3个分类）：约 5-10 分钟
   - 完整爬取（26个分类）：约 2-4 小时

3. **断点续传**
   - 爬虫支持断点续传
   - 如果中断，重新运行会自动跳过已爬取的页面
   - 已爬取的数据会保存在 output/ 目录

4. **配置参数**
   - 如果网络较慢，建议增加延时时间
   - 可以运行 `make setup-config` 调整配置
   - 详细说明见 [CONFIG_GUIDE.md](CONFIG_GUIDE.md)

## 🆘 故障排除

### 问题：提取到 0 个商品

**可能原因：**
1. URL 格式错误
2. 页面结构变化
3. 页面加载未完成

**解决方案：**
1. 检查 URL 是否正确
2. 手动打开 URL 确认商品显示
3. 增加页面加载等待时间

---

### 问题：频繁超时

**可能原因：**
1. 网络不稳定
2. 延时时间太短

**解决方案：**
1. 检查网络连接
2. 运行 `make setup-config` 增加延时
3. 或手动编辑 `playwright_crawler/config.py`

---

### 问题：分类名称显示为英文

**说明：**
这是正常的，因为 CSV 文件中的一级分类和二级分类使用的是中文名称（如 "Boots"、"高跟鞋"），这是数据来源的原始命名。爬虫会原样保存这些名称。

如果您想统一命名，可以编辑 CSV 文件，将名称修改为中文或英文。
